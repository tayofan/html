html() : 태그를 포함한 문장을 가져오기
	text() : 태그를 포함하지 않고 문장만 가져오기
	
	html(str) : str내용으로 변경 하기
				str의 내용중 html태그가 포함되어 있을때
				태그가 실행되어 출력된다
	text(str) : str의 내용중에 html태그가 포함여부 상관없이
				모두 무조건 문자(text)로 출력된다.
	
	$(선택자).html() : 
	  선택자를 대상으로 태그를 포함한 문장을 가져오기
	  선택자가 여러개인 경우 첫번째 선택자만 대상으로 한다.
	
	$(선택자).text() : 
	  선택자를 대상으로 태그를 포함하지 않고 문장만 가져오기
	  선택자가 여러개인 경우 모든 선택자를 대상으로 한다.
